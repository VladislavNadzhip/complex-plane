# Complex Plane

Настольное приложение для визуализации функций комплексного переменного — Manim-style: domain coloring, конформные преобразования, анимированные переходы между шагами, 3D-график фазы/модуля, экспорт в PNG.

## Скачать и установить

Готовые установщики для Windows / macOS / Linux собираются автоматически и лежат на странице релизов:

**→ [github.com/VladislavNadzhip/complex-plane/releases](https://github.com/VladislavNadzhip/complex-plane/releases)**

Выберите файл под свою систему:

| Система | Файл |
|---|---|
| Windows | `Complex Plane Setup <версия>.exe` — обычный установщик, ставит в меню «Пуск» и создаёт ярлык |
| macOS | `Complex Plane-<версия>.dmg` — открыть и перетащить приложение в `Applications` |
| Linux | `Complex Plane-<версия>.AppImage` (запускать напрямую, `chmod +x` при необходимости) или `.deb` (`sudo dpkg -i ...`) |

После установки приложение запускается как обычная программа — по значку на рабочем столе / в меню «Пуск», второй раз ничего собирать не нужно.

## Запуск из исходного кода (для разработки)

Нужен [Node.js](https://nodejs.org) 20+.

```bash
npm install       # установить зависимости
npm run dev       # запустить как веб-страницу (Vite) на http://localhost:5173
```

Или сразу как desktop-приложение в Electron:

```bash
npm run start      # dev-режим: Vite + Electron вместе, с горячей перезагрузкой
npm run start:prod # собрать прод-бандл и открыть его в Electron
```

### Собрать свой установщик

```bash
npm run dist
```

Готовый файл появится в `release/`. На Windows для сборки установщика (`electron-builder`) нужны права на создание symlink — либо запускать из-под администратора, либо один раз включить «Режим разработчика» в Параметры → Для разработчиков.

## Возможности

- **Преобразования** — собирайте цепочку конформных отображений (сдвиг, поворот, инверсия, Мёбиус, степень и т.д.) и смотрите, как деформируется координатная сетка/область.
- **Функция f(z)** — введите произвольное выражение (`sin(z)`, `exp(z)`, `1/z`, …) и выберите способ отображения: domain coloring, `|f(z)|`, `Re`, `Im`, `arg`.
- **Режимы просмотра** — Domain, Контуры, Кривая, Сетка, Фаза 3D.
- **Анимация** — плавные Manim-переходы между шагами преобразования (в том числе настоящие rate-функции Manim: smooth, rush in/out), с настраиваемой длительностью и зацикливанием.
- **Управление видом** — колёсико мыши — зум (с плавным инерционным приближением), перетаскивание — пан (с доводкой по инерции), двойной клик — сброс вида.
- **Экспорт** — кнопка PNG сохраняет текущий кадр вместе с осями.

## Технологии

Vite + TypeScript, Three.js (рендер), GSAP (анимации), KaTeX (формулы), Electron (desktop-обёртка), electron-builder (установщики), GitHub Actions (кросс-платформенная сборка релизов).
